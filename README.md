# Banánovník

Flashcard/kvízová aplikace pro procvičování testů.

## Online verze

Online verze dostupná na: https://tomekson.github.io/bananovnik/

## Lokální spuštění

```bash
python3 -m http.server 8080 --bind 127.0.0.1
```

Otevři: **http://127.0.0.1:8080**

> Pozor: `file://` protokol nefunguje (fetch blokován prohlížečem). Vždy spouštěj přes HTTP server.
